# 🍛 Smart Meal Planner (Streamlit App)

Smart Meal Planner is an interactive web application built using **Python** and **Streamlit** that generates personalized weekly meal plans based on:

- Diet preference (Veg / Non-Veg / Both)
- Nutrition goals (Healthy / Cheap / High Protein)
- Weekly budget
- Meal category selection

The system analyzes food nutrition data and automatically creates optimized meal schedules for the entire week.

---

## ✨ Features

✅ Weekly automated meal planning  
✅ Budget-aware food selection  
✅ Nutrition-based scoring algorithm  
✅ Veg & Non-Veg filtering  
✅ Daily calorie and protein analysis  
✅ Downloadable meal plans (CSV)  
✅ Interactive visual analytics dashboard  

---

## 🧠 How It Works

The planner uses a scoring algorithm that evaluates foods based on:

- Calories
- Protein content
- Price
- User preference goals

Meals are selected while respecting budget constraints and avoiding repetition.
